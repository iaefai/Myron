Myron by iaefai

A windowing and graphics toolkit library.

Myron contains a Windowing library supporting Windows and MacOS for the purposes of doing OpenGL work. It is intended to replace GLUT and be a little more modern. Currently it is 60% complete. It is written to support code written in C++11. The compiler support will move where the C++11 support improves. MSVC2010 and Xcode 4.2 (Clang3.0) are the minimum base compilers at this time.

Dependencies

Windows: Visual Studio 2010 or 11. MacOS: XCode 4.2 or 4.3.
